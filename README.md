QA exercise
==============================

How to execute
--------------
1. Go to root directory (qa-exercise) in the command line and run `Bundle`

2. `cd` into the `features` directory an run `cucumber features/navigation.feature`

Framework
---------

- Ruby
- Cucumber
- Watir-webdriver
- Page-Object
